# Next.js ポートフォリオサイトの構築

ポートフォリオサイトをAstroからNext.jsに移行したときの記録です。

## 使用技術

- **Next.js 15** - App Router
- **React + TypeScript** - 型安全な開発
- **Tailwind CSS** - ユーティリティファーストCSS
- **shadcn/ui** - 美しいコンポーネント
- **Framer Motion** - スムーズなアニメーション

## 工夫したポイント

### 1. レスポンシブデザイン
モバイルファーストで設計し、すべてのセクションが小さな画面でも見やすくなるように調整しました。

### 2. ダークモード対応
`next-themes`を使用して、システム設定に応じた自動切り替えを実装。

### 3. パフォーマンス最適化
- 画像の最適化
- 静的生成の活用
- 必要最小限のバンドルサイズ

## 学んだこと

Next.js 15のApp Routerは本当に便利ですね。特にレイアウトの管理が楽になりました。
